📖 Cara Penggunaan ▼

🎯 Langkah 1: Persiapan

Login ke Coretax DJP dan buka halaman "Bukti Potong" (BPPU atau BP series), pada bagian NITKU/Identity Number: Please Select ID Place of Business Activity Sub Unit Organization

📅 Langkah 2: Pilih Masa Pajak

Pilih bulan dan tahun yang diinginkan

⬇️ Langkah 3: Mulai Download

Klik tombol "Unduh Semua Bukti Potong"

⏱️ Langkah 4: Tunggu Proses

Download otomatis dengan jeda antar file

💡 Tips Penting ▼

🔄 Cara Reload Ekstensi Manual

Jika ekstensi tidak merespon atau error:

  1. Buka chrome://extensions di tab baru
  2. Cari "Bukpot Express" dalam daftar
  3. Klik tombol 🔄 Reload (segitiga melingkar)
  4. Kembali ke halaman Coretax dan coba kembali

⏹️ Cara Menghentikan Download

Jika download sedang berjalan dan ingin dihentikan:

  1. Cara 1: Klik tombol STOP Download di sidebar ini
  2. Cara 2 (Jika tombol STOP tidak berfungsi):
  3.    • Tutup browser Chrome sepenuhnya
  4.    • Download akan otomatis terhenti
  5.    • Buka kembali browser dan lanjutkan jika diperlukan

⚠️ Catatan: Menutup browser adalah cara terakhir untuk menghentikan proses yang tidak merespon tombol STOP.

🔧 Solusi Masalah Umum

  • Extension tidak merespon: Reload ekstensi secara manual
  • Download macet: Stop lalu mulai kembali
  • Halaman error: Refresh halaman Coretax dan coba lagi
  • Permission denied: Reload ekstensi dan refresh halaman
