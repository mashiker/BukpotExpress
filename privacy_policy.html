Kebijakan Privasi Bukpot Express

Komitmen Privasi Kami: Di Bukpot Express, kami berkomitmen untuk melindungi privasi Anda. Ekstensi kami dirancang untuk tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda ke server eksternal.

1. Informasi yang Tidak Kami Kumpulkan

Bukpot Express TIDAK mengumpulkan data pribadi berikut:

  • Informasi identitas pribadi (nama, alamat, email, usia, nomor identitas)
  • Informasi kesehatan
  • Informasi keuangan dan pembayaran
  • Informasi autentikasi (password, kredensial, PIN)
  • Komunikasi pribadi (email, SMS, pesan chat)
  • Informasi lokasi (alamat IP, koordinat GPS)
  • Histori browsing Anda di luar portal CoreTax DJP
  • Aktivitas pengguna di luar fungsi ekstensi
  • Konten situs web selain dokumen pajak yang Anda unduh

2. Cara Kerja Ekstensi

Bukpot Express beroperasi dengan prinsip berikut:

  • Lokal Saja: Semua pemrosesan terjadi di browser Anda secara lokal
  • Tanpa Server Eksternal: Tidak ada data yang dikirim ke server kami
  • Akses Terbatas: Hanya berfungsi di domain resmi CoreTax DJP
  • Otomasi Saja: Hanya mengotomatisasi proses download yang biasanya dilakukan manual

3. Data yang Disimpan Secara Lokal

Ekstensi ini hanya menyimpan data secara lokal di browser Anda:

  • Preferensi Pengguna: Tema tampilan (gelap/terang)
  • Log Sementara: Status proses download untuk monitoring
  • State Sementara: Informasi proses yang sedang berjalan

Semua data ini disimpan menggunakan Chrome Storage API dan tetap ada di perangkat Anda. Data dapat dihapus kapan saja melalui pengaturan browser.

4. Izin yang Diperlukan dan Alasannya

Bukpot Express memerlukan izin berikut untuk berfungsi:

  • activeTab: Untuk berinteraksi dengan halaman CoreTax DJP yang aktif
  • scripting: Untuk mengotomatisasi proses download dokumen
  • sidePanel: Untuk menampilkan antarmuka pengguna
  • storage: Untuk menyimpan preferensi pengguna secara lokal
  • tabs: Untuk memverifikasi halaman CoreTax DJP yang valid
  • Host Permissions: Hanya untuk domain resmi CoreTax DJP

5. Keamanan Data

Kami menjaga keamanan dengan:

  • Tidak mengumpulkan data pribadi sama sekali
  • Semua operasi terjadi secara lokal di browser
  • Tidak ada komunikasi dengan server eksternal
  • Source code terbuka yang dapat diaudit

6. Kepatuhan Regulasi

Bukpot Express mematuhi:

  • Kebijakan Privasi Chrome Web Store
  • Regulasi perlindungan data yang berlaku
  • Praktik privasi terbaik untuk ekstensi browser

7. Perubahan Kebijakan Privasi

Jika kami melakukan perubahan pada kebijakan privasi ini, kami akan:

  • Memperbarui tanggal "Terakhir Diperbarui" di dokumen ini
  • Menyediakan versi baru kebijakan privasi
  • Menginformasikan perubahan signifikan melalui deskripsi ekstensi
